# ST-MIND 2.0 - Research & Evaluation

## 1. Overview
As a platform for researchers as well as educators, ST-MIND 2.0 must capture high-quality, privacy-preserving data on student engagement and AI efficacy.

## 2. RAG Performance Metrics
- **Retrieval Accuracy:** Are the correct document chunks retrieved for a given query?
- **Answer Relevance:** Does the generated answer directly address the student's question?
- **Citation Accuracy:** Does the answer accurately reflect the cited text without distortion?

## 3. User Engagement Metrics
- **Question Frequency:** Volume of questions asked per course / per document.
- **Feedback Distribution:** Ratio of Helpful / Incorrect / Unsupported flags.
- **Resolution Rate:** Do students ask follow-up questions, or does the first answer suffice?

## 4. Privacy & Anonymization
- All data exported for research must be strictly anonymized.
- PII (Personally Identifiable Information) must be stripped from query logs before analysis.
- Explicit consent mechanisms will be implemented for research data usage.
